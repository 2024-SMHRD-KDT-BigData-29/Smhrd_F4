## 👀 서비스 소개
* 서비스명:  Work(일)와 Clean(청결)을 결합한 Worklean
* 서비스설명: AI 기반 이상치 탐지 모델을 적용한 실시간 공기질 모니터링 시스템으로, 이상이 감지되면 HVAC(공조) 장비를 자동 제어합니다.
